Table of Powers

Instructions

Write a program that prints out a table of exponential numbers. This assignment is similar to the previous Table of Cubes assignment, but has a varying number of columns per row (and so requires nested loops). Prompt the user for the number of rows and the highest power. Note that the first value in each row is right-aligned in two spaces and every other value is right-aligned in eight spaces. You will need to find the setw() function useful to control the spacing.
